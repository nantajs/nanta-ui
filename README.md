# nanta-ui
ant-design based ui framework.

## Deps

- Vue 3.0  
- [ant-design-vue](https://github.com/vueComponent/ant-design-vue)

## Notice
nanta-ui is wip, current alpha stage.

## How to dev
1. build package
```
yarn package
```

2. test pkg in local
```
yarn pack  # pack local
npm install nanta-ui-0.0.1.tgz  # install local
```

3. publish to npm
